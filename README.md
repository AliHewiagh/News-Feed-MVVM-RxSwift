# News-Feed-MVVM-RxSwift

> 


## Requirements

- iOS 13.0+
- Swift 4

## Getting Started

1. Download the latest Xcode
2. Clone this repository
3. Install [Cocoapods](https://cocoapods.org)
4. Open Cinemagic.xcworkspace in Xcode


#### Third Party Libraries

1. SnapKit
2. RxSwift
3. RxCocoa
4. SwiftyJSON
